# 美妆类目水印规范

## 主图水印

- **位置**：主图右下角，距右边 80px、距底边 80px
- **透明度**：50%（半透明，不抢主体）
- **字体**：PingFang SC Regular，字号 24px
- **颜色**：白色，必要时加 1px 黑色描边

## 详情页水印

- **位置**：详情页左上角，距左边 40px、距上边 40px
- **透明度**：30%（更弱，仅作品保护）
- **字体**：PingFang SC Light，字号 16px
- **颜色**：白色

## 审核 SOP

1. 主图 + 详情页都需加水印（白底图、场景图均需）
2. 模特脸部不应被水印覆盖
3. 文字不应与水印重叠
4. 不在 SKU 大促主图加品牌角标（已与运营对齐）

## 例外

- 跨境业务 SKU 不加水印（运营单独通知）
- 节日主题图（圣诞、新年）走特殊审核流
