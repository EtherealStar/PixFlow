# PixFlow Web Design Entry Point

PixFlow Web 的权威设计位于 [`frontend/`](./frontend/README.md)：

- [`frontend/product.md`](./frontend/product.md) 定义用户可见产品行为；
- [`frontend/api.md`](./frontend/api.md) 定义前端依赖的后端契约；
- 其他 `frontend/*.md` 按 Shell、Chat、Materials/Outputs、Activity、Upload、Transport、Runtime 和 UI 拆分模块边界。

本文不再重复状态机、旧占位路由、历史审计结果或实现进度。若代码、旧文档片段与 `frontend/` 冲突，以 `frontend/` 与对应后端模块设计为准。
